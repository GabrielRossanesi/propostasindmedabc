# Bloom Gifts Franchise Portal - Proposta Comercial

Proposta comercial interativa em Next.js para apresentar o Bloom Gifts Franchise Portal, uma plataforma B2B privada para pedidos, aprovação, pagamento e produção de brindes corporativos em redes de franquias.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Comandos

```bash
npm.cmd install
npm.cmd run dev
npm.cmd run build
```

## Estrutura

- `src/app`: layout, estilos globais e rota principal.
- `src/components`: componentes reutilizáveis da proposta.
- `src/lib`: dados estruturados do conteúdo comercial.
